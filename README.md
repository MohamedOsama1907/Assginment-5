# Assignment-5 – Route Academy (💰 Mudabbir Financial Dashboard)

A responsive financial dashboard built using HTML5 & CSS3, designed to manage personal finances and track expenses through a clean, structured UI.
## 📌 Features

* Fully responsive dashboard layout (Mobile → Tablet → Desktop)
* Built using CSS Grid & Flexbox
* Sidebar navigation with structured dashboard layout
* Summary cards for financial insights
* Styled tables for transactions tracking
* Budget overview with progress indicators
* 3D credit card design using CSS transforms
* Smooth animations and transitions for better UI experience

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome (icons)
* Google Fonts
* CSS Variables (Custom Properties)
---

## 🎯 What I Practiced

* Using CSS **Grid & Flexbox** to build complex layouts
* Creating a full dashboard UI from scratch
* Implementing CSS animations and transitions
* Building 3D card effects using transform (3D)
* Writing reusable and maintainable CSS using variables
* Implementing responsive design with media queries
* Styling advanced UI components (cards, tables, progress bars)
* Applying pseudo-elements and hover effects

---

## 📱 Responsive Breakpoints

* Mobile: Default
* Tablet: `min-width: 767px`
* Desktop: `min-width: 1024px`
* Large Screens: `min-width: 1370px`
